# kwin-FocusLock
KDE plasma 6 wayland kwin script



Locks the amount of times the user can switch windows in a give time.
The amount of switches allowed in the form of a cap, e.g. 8 maximum switches
Is controlled by a switch regeneration time, where 1 switch regens per alloted time.

A shortcut key is available to set which will give a switch when pressed.
